# 天穹安防系统 - 网络安全智能分析平台

## 项目概述

天穹安防系统是一款综合性的网络安全智能分析平台，集成了入侵检测系统(IDS)、态势感知、自动防御、病毒查杀、应急响应等核心安全功能，旨在为企业和组织提供全方位的网络安全监控与防护能力。

## 核心功能特性

### 安全监控与检测
- **入侵检测系统(IDS)**: 实时监控网络流量，检测异常行为和潜在威胁
- **基线排查**: 定期检查系统配置基线，确保符合安全标准
- **态势感知**: 可视化展示网络安全态势，提供威胁情报融合分析

### 防御与响应
- **自动防御**: 根据检测结果自动执行防御措施，阻止攻击
- **病毒查杀**: 扫描文件和系统，检测并清除恶意软件
- **应急响应**: 提供安全事件响应流程和工具，快速处理安全事件

### 系统管理与优化
- **用户权限管理**: 基于角色的访问控制，确保系统安全
- **日志记录**: 详细记录系统操作和安全事件日志，便于审计和分析
- **性能监控**: 实时监控系统性能，优化资源使用
- **安全审计**: 定期执行安全审计，确保系统合规性

## 技术栈

- **前端**: HTML5, JavaScript, CSS
- **后端**: 轻量级API服务
- **安全组件**: 入侵检测引擎、病毒特征库、威胁情报集成
- **自动化**: CI/CD流程(GitHub Actions)
- **测试**: 自动化测试框架

## 快速开始

### 安装与配置

1. **环境要求**
   - 支持现代浏览器(Chrome, Firefox, Safari, Edge)
   - 服务器环境(Node.js 16+ 或其他Web服务器)
   - 推荐操作系统: Linux(生产环境), Windows/MacOS(开发环境)

2. **部署步骤**
   ```bash
   # 克隆项目
   git clone [repository-url] xuan-guang-security-gpt
   cd xuan-guang-security-gpt
   
   # 安装依赖(如果需要)
   npm install
   
   # 启动服务
   # 方式1: 使用Node.js静态服务器
   npm start
   # 方式2: 使用其他Web服务器(如Nginx)部署web目录
   ```

3. **配置说明**
   - 访问系统配置页面(`/config`)
   - 配置API密钥、代理设置等
   - 设置扫描参数和安全策略

### 使用指南

1. **系统登录**
   - 默认登录页面: `/login`
   - 初始管理员账号: `admin` (首次登录需设置密码)

2. **主要功能入口**
   - 仪表盘: 概览系统状态和安全事件
   - 入侵检测: 配置和管理IDS规则
   - 态势感知: 查看网络安全态势图
   - 日志记录: 查看系统和安全事件日志
